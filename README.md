## 1. 🏓 乒乓球遊戲 AI 系統

本專案的核心是使用 **Deep Q-Network (DQN)** 強化學習演算法，訓練一個 AI 代理來學習並掌握乒乓球遊戲的策略。

### 核心技術

-   **深度強化學習 (DRL)**：結合深度學習與強化學習，處理高維度狀態空間。
-   **Bellman 方程應用**：用於計算最優動作價值函數。
-   **經驗回放 (Experience Replay)**：儲存並隨機取樣經驗，打破數據相關性，提高訓練穩定性。
-   **目標網絡 (Target Network)**：使用獨立的目標網絡來穩定 Q 值更新，防止訓練發散。
-   **完整的測試與驗證流程**：確保 AI 模型的性能和穩定性。

### 目標與評估指標

我們設定了以下關鍵指標來評估乒乓球 AI 系統的性能：

| 指標 | 目標值 |
| :----------- | :-------- |
| 接球成功率 | ≥85% |
| 推理延遲 | <30 ms |
| 訓練收斂時間 | ≤2 小時 |

## 2. 🎮 TetrAI - 俄羅斯方塊對戰 AI (未來規劃)

此為一個進階的未來項目，目標是開發一個使用多代理強化學習訓練的俄羅斯方塊 AI，使其能夠與人類玩家進行競爭。

**預計功能與技術：**

-   **進階機制支援**：支援 B2B (Back-to-Back)、T-Spin、Tetris、Ghost Piece 等複雜遊戲機制。
-   **Socket Server 架構**：採用 Processing (遊戲前端) + Python (AI 後端) 的 Socket Server 通訊架構。
-   **混合策略**：結合遺傳算法 (Genetic Algorithms) 與 DQN 強化學習策略。
